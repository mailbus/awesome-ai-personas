# 贡献指南 | Contributing Guide

感谢你的贡献！Thank you for your contribution!

## 简体中文

### 如何贡献

1. **Fork 本仓库**
2. **创建分支**: `git checkout -b add-new-project`
3. **编辑 README.md**
4. **提交更改**: `git commit -m "Add: 项目名称"`
5. **推送分支**: `git push origin add-new-project`
6. **创建 Pull Request**

### 添加新项目格式

在相应分类下添加：

```markdown
| [用户名/项目名](https://github.com/username/repo) | [![Stars](https://img.shields.io/github/stars/username/repo?style=social)](https://github.com/username/repo) | 简短描述（中文） |
```

### 分类标准

- **人格角色 Personas**: 模拟特定人物角色的知识、习惯、行为模式
  - 工作关系: 同事、老板、导师
  - 家庭关系: 父母、伴侣
  - 情感记忆: 前任、重逢
  - 自我探索: 自己、数字分身

- **智能工具 Tools**: 增强 AI 能力的实用工具
  - 安全防护: 防蒸馏、隐私保护
  - 效率工具: 提效、自动化

### 要求

- ✅ 项目必须已发布在 GitHub 上
- ✅ 与 AI 人格或智能工具相关
- ✅ 提供**中英双语**描述
- ✅ 按格式正确添加 Stars 徽章

## English

### How to Contribute

1. **Fork this repository**
2. **Create a branch**: `git checkout -b add-new-project`
3. **Edit README.md**
4. **Commit changes**: `git commit -m "Add: Project Name"`
5. **Push branch**: `git push origin add-new-project`
6. **Create Pull Request**

### Adding New Projects

Add to the appropriate category:

```markdown
| [username/repo](https://github.com/username/repo) | [![Stars](https://img.shields.io/github/stars/username/repo?style=social)](https://github.com/username/repo) | Brief description (English) |
```

### Category Standards

- **Personas**: Simulate knowledge, habits, and behavioral patterns of specific roles
  - Workplace: Colleagues, bosses, mentors
  - Family: Parents, partners
  - Emotional: Ex, reunion
  - Self-Exploration: Self, digital twins

- **Tools**: Practical tools that enhance AI capabilities
  - Security: Anti-distillation, privacy protection
  - Productivity: Efficiency, automation

### Requirements

- ✅ Project must be hosted on GitHub
- ✅ Related to AI personas or intelligent tools
- ✅ Provide **bilingual** descriptions (Chinese & English)
- ✅ Add Stars badge in correct format
