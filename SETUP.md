# 快速启动 | Quick Start

---

## 简体中文

### 一键创建仓库

```bash
# 创建本地目录
mkdir awesome-ai-personas
cd awesome-ai-personas

# 初始化 Git 仓库
git init
git branch -M main

# 复制已创建的文件到当前目录后：
git add .
git commit -m "Initial commit: Awesome AI Personas list"

# 连接到 GitHub（替换 YOUR_USERNAME）
git remote add origin https://github.com/YOUR_USERNAME/awesome-ai-personas.git
git push -u origin main
```

### GitHub 上创建仓库步骤

1. 访问 [GitHub](https://github.com) 并登录
2. 点击右上角 `+` → `New repository`
3. 仓库名称填写：`awesome-ai-personas`
4. 设置为 **Public** 公开仓库
5. **不要**勾选 "Add a README file"
6. 点击 `Create repository`
7. 按照页面提示推送本地代码

### 添加 GitHub Topics

在仓库设置中添加以下标签以提高发现率：
- `ai-personas`
- `awesome-list`
- `agent-skills`
- `ai-agents`
- `chatgpt`
- `llm`

## English

### One-Command Repository Creation

```bash
# Create local directory
mkdir awesome-ai-personas
cd awesome-ai-personas

# Initialize Git repository
git init
git branch -M main

# After copying created files to current directory:
git add .
git commit -m "Initial commit: Awesome AI Personas list"

# Connect to GitHub (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/awesome-ai-personas.git
git push -u origin main
```

### Steps to Create Repository on GitHub

1. Visit [GitHub](https://github.com) and log in
2. Click `+` → `New repository` in top right
3. Repository name: `awesome-ai-personas`
4. Set as **Public**
5. **Do not** check "Add a README file"
6. Click `Create repository`
7. Follow the prompts to push your local code

### Add GitHub Topics

Add these topics in repository settings for better discoverability:
- `ai-personas`
- `awesome-list`
- `agent-skills`
- `ai-agents`
- `chatgpt`
- `llm`
