## 版本三：技术版（面向开发者）

> 这些 Skill 项目本质上是对"人格化 Prompt"的结构化沉淀。我整理了一个 Awesome List，并做了一个 **Agent 安装指南**：
>
> **https://github.com/mailbus/awesome-ai-personas**
>
> **设计思路：**
> - 将每个 Skill 视为独立的能力包，可组合使用
> - AGENT_GUIDE.md 用结构化格式编写，AI Agent 可以解析并执行批量安装
> - 支持按分类安装（工作关系、情感记忆、自我探索等）
>
> **使用示例：**
> ```
> # 发给 AI Agent
> "请根据 AGENT_GUIDE.md 安装所有工作相关的 Skills"
> ```
>
> AI 就会自动克隆：colleague-skill, boss-skills, supervisor, senpai-skill
>
> 这个模式可以扩展：未来可以把任何领域知识封装成 Skill 包，形成开放的"人格能力市场"。
>
> 欢迎大家一起完善这个列表~ 🚀
